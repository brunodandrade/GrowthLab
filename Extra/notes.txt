
GrowthLab

Calculadora Base:

Investimento inicial, aportes mensais, taxa de juro, período
Diferentes frequências de capitalização (mensal, trimestral, anual)
Cálculo com e sem inflação ajustada
Impostos sobre ganhos de capital

Inputs Dinâmicos:

Sliders interativos com valores que se atualizam em tempo real
Validação de dados (números positivos, limites realistas)
Formatação automática de moeda (€, $, etc.)
Presets comuns (reforma, casa própria, emergência)

Visualização de Dados
Gráficos Interativos:

Linha temporal mostrando crescimento ao longo dos anos
Gráfico de barras: principal vs. juros vs. aportes
Gráfico circular: composição final do investimento
Zoom, hover effects, tooltips informativos

Tabelas Detalhadas:

Breakdown ano a ano com valores específicos
Export para CSV/Excel
Possibilidade de expandir/colapsar detalhes mensais

Features Avançadas
Cenários Múltiplos:

Comparação lado-a-lado de diferentes estratégias
"E se?" scenarios (mudança de taxa, aportes variáveis)
Simulação de crises (anos com retorno negativo)
Diferentes perfis de risco

Calculadoras Especiais:

Meta reversa: "preciso de X€, como chegar lá?"
Reforma: quando consegues parar de trabalhar
Inflação personalizada por país/região
Custos de oportunidade

Interface e UX
Design Profissional:

Dark/light mode toggle
Layout responsivo (mobile-first)
Animações suaves nas transições
Loading states para cálculos pesados

Usabilidade:

Tooltips explicativos para termos financeiros
Undo/redo para mudanças rápidas
Guardar configurações no localStorage
URL partilhável com parâmetros

Fórmulas matemáticas precisas
Validação robusta de inputs
